Activity: Travis CI Setup

We're going to use my testing demo repo as a starting point.

  • Create a personal access token. Github Token Settings
  • Copy that token, and then go to the Travis CI build settings.
  • Create an environment variable with the key GH_TOKEN whose value is the token you just copied. This allows Travis to do the push to the branch to the gh-pages branch every time the build passes.
  • On line 14 of the deploy script change the email to your github email.
